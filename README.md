# BHBChatRoomBG

## 简介

这是一个给BHB聊天室添加图片（外链）的插件。

这个插件按照W3标准编写，对Chrome的兼容由[polyfill](https://github.com/mozilla/webextension-polyfill)提供。

## 使用方法

点击插件，在弹出的页面设置链接和文本，点击Save保存即可。
